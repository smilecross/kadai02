作りたいもの第一弾：スマートライフ診断
【どんなサービス】
・デジタル社会にシフトしていく中で、スマホを暮らしに活用できるようになるために、まずは自分の現在地をするための診断サービス

【目指すゴール】
・カテゴリーごとに点数評価し、診断結果をチャートで表示する

【マイルストーン】
1.診断結果をテキストで表記する
2.診断結果ごとに異なるモーダル表示をする
3.問題数を増やし、ランダムに10問を抽出して問題を出す
4.回答の選択肢をランダムに表示する
5.診断結果をチャートで表示する
6.メールアドレスを登録して、結果を送信する
